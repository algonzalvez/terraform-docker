*** TERRAFORM - DOCKER ***
> Uso de Terraform para orquestar contenedores de Docker. La finalidad es desplegar dos servidores nginx con un index personalizado.

## Iniciar proyecto terraform

```init
terraform init
```

## Comprobar sintaxis

```Validate
terraform validate
```

## Realizar el plan

```plan
terraform plan
```

## Desplegar proyecto

```apply
terraform apply
```
